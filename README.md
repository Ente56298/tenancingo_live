# Tenancingo Live - Sistema de Apuestas en Vivo

## 🎮 Descripción
Sistema completo de apuestas en línea con transmisión en vivo y panel administrativo.

## 🚀 Características
- Sistema de autenticación seguro
- Panel de usuario y administrador
- Juegos de apuestas en tiempo real
- Gestión de saldo y transacciones
- URLs amigables
- Diseño responsive

## 📋 Requisitos
- PHP 7.4+
- MySQL 5.7+
- Apache con mod_rewrite

## ⚙️ Instalación
1. Clonar repositorio
2. Configurar base de datos en config.php
3. Importar estructura SQL
4. Configurar permisos de archivos

## 🔗 Demo
- Sitio: https://tenancingolive.byethost17.com/
- Admin: /admin
- Login: /signin
